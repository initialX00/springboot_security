ㅇ도커란
리눅스os 기반 프로그램이다. (git bash도 리눅스 기반이다)

pc의 os 설치,프로그램 설치,프로그램 설정 등 환경을 그대로 카피한 .iso이미지를 활용한 프로그램이다.

도커로 이미지를 만들거나 실행하며 도커hub로 공유를 한다.

다른 프로그램과 달리 딱 실행만 가능할 정도의 linus운영체제가 포함되어 가볍다.
linus운영체재를 사용하여 별도의 운영체재로 실행되는 가장pc이기에, 기존pc에 사용되는 프로그램과 별개로 사용할 수 있다.
컨테이너를 적제하는 개념으로 필요한 최소한의 정보만 담기에 가볍다. 

포트번호6379
docker pull redis mysql 이미지 다운.
docker run name rs -p6379(외부:내컴퓨터) : 6379(내부:도커) -d redis
도커의 컨테이너 자체로는 내 pc에 저장할 수 없지만, 외부와 내부를 연결하여 사용한다. (외부 컨테이너를 내부 컨테이너에 저장)
